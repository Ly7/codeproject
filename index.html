Why I choose to learn coding with Code First Girls


I got stuck - a lot - when I tried it on my own. I tried to learn it through websites and free online course - not so easy if don't know where to start.

So now, with Code First for Girls, if I get stuck, they help me out. And you know what? It sometimes is a very small mistake like forgetting to close a div or a double space instead of a single one.

I love the course as it is simple and easy to understand and encourages you to create your own website. I never thought I could have my own website within a few weeks, but here I go. Join the fun - It is for free!!!!

How does it work?


  • There are 7 one-hour weekly sessions where you are taken through the course content and then you will use the newly learned code in several exercises. This helps you understand how coding language works and how it is to be used.

  • After the session, there is some at home practice which shouldn't take more that 4 hours a week. This helps you anchor what you have learned and you can implement it on your own website.

  • You would need to attend all sessions as they are all linked together and you may not understand parts of it otherwise.


What will I learn?


  • week 1: What is a web page? a web server? How to create a HTML page.

  • week 2: How to use GIT to get my page on the web? WHat is CSS? How do I use CSS with HTML? What are CSS attributes? Publish your first website now!

  • week 3: What is Twitter Bootstrap? How does it help me to design websites? How do I use and modify it?

  • week 4: THE Project! Show us what you can do already!
